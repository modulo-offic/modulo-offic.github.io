Как скачать

Определимся сразу: при создании программы мы были воодушевлены идеей оптимизации работы библеотекаря. Мы представляем вашему вниманию программу, которая будет хранить в себе имена учеников и списки учебников, которые они взяли, а также количество книг, которые можно взять в библиотеке, должников и многое другое. Базовая версия программы распространяется бесплатно, в ней есть все необходимые функции для комфортной работы. Но есть и ПРО версия для тех, кто хочет максимально упростить свою работу и убрать все ограничения. О всех функциях базовой и ПРО версии будет рассказано ниже.

Установка

Обратите внимание!

Каждый может установить нашу программу бесплатно!

Порядок установки базовой версии

Нажмите кнопку скачать. По окончании загрузки нажмите на файл с разрешении .exe, чтобы начать загрузку. Файл автоматически начнёт загрузку в нужное место. После установки появится ярлык программы на рабочем столе. Запускаем и наслаждаемся. Путь к файлам программы по умолчанию: C:\...\...\AppData\Local\Programs\Min_lib_2020 .

Порядок установки ПРО версии
